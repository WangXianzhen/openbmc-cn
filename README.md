## 🎯 关于我们

**OpenBMC 中文技术站** 致力于为国内 BMC 开发者提供：

- 📚 系统化的入门教程
- 🔧 深入的实战案例
- 📖 协议规范的解读
- 🛠️ 常用工具的使用指南

## 🛠️ 技术栈

- **固件框架**: OpenBMC (Linux Foundation 项目)
- **构建系统**: Yocto Project / Bitbake
- **通信协议**: IPMI 2.0、Redfish、REST API
- **主要芯片**: ASPEED AST2600/2700、高通、英特尔
